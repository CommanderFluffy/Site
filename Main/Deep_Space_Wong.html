Features
Deep Space Wong is an arcade style space shooter.
It features: Randomly generated terrain, Boss fights, powerups, unlockables, different AI difficulty, different gamemodes, an awesome soundtrack from Kubbi and some sprites from George.









About

Deep Space Wong was my first decent 2D game

It’s actually quite a hard game, especially the “OMG” mode. A friend of mine actually got something like 2000, see if you can beat it.

Download
Deep Space Wong comes as a .jar, just double click to launch it.

It will black screen for a few seconds when you start it as I messed up startup so I’m loading three songs into memory before I display the splash screen.

12.6 megabytes download size
