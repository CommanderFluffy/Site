Features
Pokewon Alpha is a text based adventure.
Its features include: combat system with equip-able items, monsters, full interaction with items, many commands, an interesting plot and lots more.

Pokewon is able to understand commands such as “Go west” as well as more complex commands such as “take icecream from fridge”










About

Pokewon was released in April of 2012, it runs in the terminal of the operating system.

Pokewon is still in the Alpha stage, it was going to be a full fledged text based pokemon clone hence the name but I eventually stopped development, it does however support combat with pokemon in the combat system.


Controls

Type text commands
Make sure to read the list of commands before playing
