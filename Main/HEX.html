Features
This is a project that I did for a course in UNI. It was supposed to be a group project where teams of three work together to produce a solution, but these sorts of projects tend to have slackers and in my case I did all the work. However I still managed to produce a decent little game in the end.


The main objective of the game is to reach the goal tile in the least number of moves, you can choose to have up to 4 players and up to 4 playing pieces.


The help screen pretty much explains everything else.


It also has a number of different themes, which I like to add to mu GUIs.










































































































Download
This comes as a .jar, just double click to launch it.

Note: this works on OSX, Windows, and Linux.
