Info
For those that don’t know what a raycaster is, it is a psudo 3d engine. That means it uses 2d maths to render a seemingly 3d scene. Old games like wolfenstein 3d used raycasting to render their levels!


Features

My raycaster currently has textured walls, floors and ceilings. It is also shaded. Plans for the future include sprites and three discrete levels but that looks like a pain in the neck to implement so maybe not.

Currently uses Minecraft textures because out of the ones I tried they looked the best.
