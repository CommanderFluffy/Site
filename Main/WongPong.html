Features
WongPong is a basic pong clone. It features some graphic options, different AI difficulties and different game modes including a multiplayer option.











About

WongPong was the first game I wrote all the way back in september 2011.

WongPong is written in Java with the Core Processing library.


Controls

Up - ‘Z’                    Player2 up - ‘<’

Down - ‘X’                Player2 down - ‘>’

Enter - SPACE


Download
WongPong comes as a zip, just unzip it and run WongPong.exe
