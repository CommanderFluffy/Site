Features
This is a timetable manager which is kind of like a calendar, but this is written with our timetable in mind.
Stuff it can do: scrollable date bar, correct information about the day and time, shows your periods, image tiles with subjects, 13 different colour schemes (only a few shown here) and a nice looking interface.


A really cool feature are the Reminders, they are smart. You can set up a reminder for a particular subject and because it knows your timetable it will let you choose from your actual time table lessons so you don’t have to enter the date or the time. It will do it for you!


There are a few more minor features that show useful information in the interface.


The program is almost finished, it just needs the functionality to edit timetables in a user friendly interface.


Instructions

Once the program has loaded, click the arrow to progress.
At the main menu, click the settings cog to set the interface colour and current week, then return to the main menu.

Now click on the timetable icon and add all your subjects using the plus icon. Once this is done click the cog to go into edit mode.
Select a subject from the right hand list, once a subject is selected you can then click on a square in the timetable to set that period to be the selected subject. Once you are done click the tick to save or the cross to cancel, then repeat the process with the other week, you can navigate between weeks using the arrows at the top.

Now you are ready to use the calendar, press the icon labeled day view to change to that view.

Here you are presented with an overview of your day, you can change days by selecting different ones and scrolling through the list.
The hourly view will show you your reminders, clicking on a reminder will show you more information.

The plus icon will allow you to set a new reminder.

The arrows help select different days.

The dot returns you to the current day.

Mousing over a picture will show you more information.

Pressing Escape will return you to the weekly view.
