Info
Prelude of a Hero is a simple side scroller.
It was originally intended to be a game in which you traveled between screens, however my friends talked me into making it into a proper side scroller so I had to adapt my original engine into something it wasn’t designed to handle so its kinda a mess... but it works.


Features (So far)

  1. -parallax background

  2. -Animated objects, such as flickering street lamps

  3. -Collision is able to detect multiple levels
