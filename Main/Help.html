OSX Users

Due to changes made by Apple to delegate the java runtime back to ORACLE and the default JRE for OSX to be 1.7 in many cases my games will no longer run.


My games use the “LWJGL” library which used libraries from Apple’s 1.6 JRE but now with Java 1.7 this library is no longer supported.



If you suspect this is why the game is not running on your computer try typing “java -version” into “Terminal”

(located in /applications/utilities/) then press enter if you get results that look like this:



















Then that is indeed the problem, if you do know how to revert back to 1.6 that will resolve the problem, if you do not I am not going to explain how. Just try it on a windows machine.
